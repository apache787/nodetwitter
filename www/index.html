Node.js Twitter API Launcher

Author: Keith Blackard

Welcome to the Node.js Twitter API Launcher! Here I've provided a quick demonstration of HTML, CSS, and Javascript working together using common Libraries including JQuery and PureCSS. You can launch AJAX events to be handled asynchronously by both the browser and the Node.js Server.

Below on the left is the Event Launchers Panel. Here are 3 quick buttons that call an AJAX event using the JQuery library. Each can be clicked in any order, and will be processed in the order the server responds back with.

Below on the right is a quick html console to show whats going on in the background, and provide links to CSV files that are generated by the server

Things to do:

  • Check page responsiveness to screen sizes
  • Click on Event buttons in various orders
  • Download csv file to inspect.
